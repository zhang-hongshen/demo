# 智教方案生成台演示话术

## 启动

```bash
npm start
```

打开 `http://localhost:5173`。

## 现场流程

1. 展示左侧需求输入：课程、主题、班级画像、学生痛点、教学目标。
2. 说明这是高校教师备课、授课、测评、补救的完整闭环，不是通用聊天问答。
3. 点击“生成智教方案”，由后端读取 `.env` 中的 `api_key` 并调用智能生成服务；完整生成通常需要30到60秒。
4. 依次展示教案、课件大纲、随堂测验、学情分析、演示话术五个标签。
5. 强调可对接教务平台、学习平台、课堂互动数据，形成校级 AI 教学应用入口。

## 备用方案

如果现场网络或智能生成服务暂时不可用，点击“载入样例”。样例内容与真实输出结构一致，可以保证竞标答辩或实地考察演示不中断。

## 推荐讲法

“老师只需要输入课程主题、班级画像和教学目标，平台就能通过智能生成能力产出一套可落地的课堂方案。对校方信息化部门来说，这个能力不是单点工具，而是可以嵌入备课、授课、测验、学情分析和课后干预的教学闭环。”
